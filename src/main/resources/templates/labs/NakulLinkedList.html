Adding and Deleting Complexity

ArrayList .add(): O(1) {Only one operation needed to add object at beginning of array list}

ArrayList .add(index, element): O(n) {List must be iterated through to find specific index for object to be inserted}

ArrayList .remove(): O(n) {List must be iterated through to find specific index for object to be removed}

LinkedList .add() and .remove(): O(n) {Takes n operations to find the node for insertion/deletion and then one operation to carry it out}
